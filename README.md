## 실행 방법
### Backend
1. backend 폴더 안에 .env파일이 있는지 확인합니다.
2. make up을 실행합니다.

### Frontend
1. cd /assetmanagement/frontend
2. sudo npm i
3. sudo npm run dev

### 코드 커밋
